# MoodTrack-ML

MoodTracker-ML is a beginner friendly machine learning web app that tracks daily mood, sleep, stress, study/work hours, exercise, screen time and water intake.

The app predicts a user's mood category as Good, Okay or Bad using a simple machine learning model.

## Features

- Track daily mood-related habits
- Store mood data in a CSV file
- Analyze mood patterns
- Train a basic ML model
- Predict mood using user input
- Visualize mood trends
- Simple Streamlit web interface

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
- Git and GitHub
